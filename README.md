Developed a comprehensive retail analytics dashboard to monitor sales performance, inventory levels, supplier efficiency, and profitability. The project analyzes category-wise sales quantity, total revenue, profit contribution, reorder levels, and supplier performance. It enables better inventory planning, profit optimization, and supplier decision-making through visually intuitive insights.
Key Focus Areas Covered:
Sales and profit analysis
Inventory and stock monitoring
Category-wise and product-wise performance
Supplier-wise profit and reorder analysis
Business summary with key financial metrics
